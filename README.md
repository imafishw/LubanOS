# luban_os


## Project introduce
**这是一个机器人终端的半成品，在网络通信接口部分还未写入**
**拥有三个主要页面**
**第一个页面用于展示机器人的一些历史数据以及介绍**
**第二个页面用于读取机器人身上摄像头传送的视频流**
**第三个页面作用于控制机器人的一些模式以及状态**
